Tabela de variáveis

A presente tabela indica as variáveis que serão trabalhadas ao longo do Projeto Integrador III. Consiste em dados de medição oceânica comumente usadas para prever o fluxo marítimo, inundações, ressacas e fenômenos similares.

O dataset foi extraído do ERA5, conjunto de dados climáticos global, programa de obtenção de imagens de satélite do European Centre for Medium-Range Weather Forecasts (ECMWF). Este instrumento fornece estimativas horárias para um grande número de grandezas atmosféricas, de ondas oceânicas e da superfície terrestre.

Para este estudo, foram coletados os dados de Vento, pressão média do mar, inclinação média das ondas e etc. para prever a variável de altura máxima das ondas ao longo da região costeira de Santos com dados de 2025.
